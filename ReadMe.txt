This is a project based on learnings from Udacity Nanodegree course of Android.
It is an app names Sunshine to view the weather forecasts for next seven days based on the location.

Please note, for this app to work, user needs to update the APP ID as below:

1)Follow https://openweathermap.org/appid and create APP ID
2)Update the ID in ForecastFragment.java file where its marked <ENTER_YOUR_APP_ID>
